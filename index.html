Conway's Game of Life

2020

The goal of this practical was to produce an implementation of Conway’s Game of Life. This implementation had a graphical interface that allowed a user to create a board configuration, and then processed it step by step using the game’s rules, showing the state of the board as it went along.

Eliza

2020

The goal of this practical was to produce an Eliza-style program of your own, in Java, with three different script files that it read from. It took input from the user on the command line, and printed output in response, mimicking a real person as closely as possible. Three personalities were made: Eliza a psychotherapist, Stuart a person offering technical support and Alex a conspiracy theorist.

Coursework

  • Object-Oriented Programming

    This module covered problem-solving skills, programming basics and object-oriented concepts, modelling and programming. The main language learnt in this module was Java. Practical skills were reinforced through a range of exercises and assignments covering these topics.

  • Programming with Data

    This module explored various aspects of data storage, processing and analysis. Programming skills were reinforced through a range of exercises and practicals covering various aspects of data handling. Topics include: persistent data formats; files and databases; file manipulation; binary and textual data; data processing using open source libraries; database design and use; object-relational mapping frameworks; processing and analysing data; issues of scale. Themes related to current research in the area of data science and big data were emphasised.

  • Programming Projects

    This module reinforced key Java programming skills gained earlier, by means of a series of coursework assignments posed as mini-projects within small groups. These projects were designed to offer increasing depth and scope for creativity as the module progressed.

  • Foundations of Computation

    This module introduced fundamental algorithms, data structures and formal language concepts at the heart of modern software, and develops skills in programming and analysis.

  • Computer Systems

    This module developed skills in programming in C, systems programming, digital logic and low-level computer organisation.

  • The Internet and the Web: Concepts and Programming

    This module introduced me to the Internet and the World Wide Web from a Computer Science perspective. It consisted of two complementary streams: computer networks and web-based computing. Both streams introduce key concepts, current technologies, programming abstractions and the practical aspects of developing web-based and network applications.

  • Advanced Programming Projects

    This module introduced the functional and dynamic programming paradigms, using languages such as Haskell and Python. Understanding was reinforced through multiple group projects.
