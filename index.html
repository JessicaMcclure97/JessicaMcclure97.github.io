Conway's Game of Life

2020

The goal of this practical was to produce an implementation of Conway’s Game of Life. This implementation had a graphical interface that allowed a user to create a board configuration, and then processed it step by step using the game’s rules, showing the state of the board as it went along.

Eliza

2020

The goal of this practical was to produce an Eliza-style program of your own, in Java, with three different script files that it read from. It took input from the user on the command line, and printed output in response, mimicking a real person as closely as possible. Three personalities were made: Eliza a psychotherapist, Stuart a person offering technical support and Alex a conspiracy theorist.
